QA Automation Portfolio – Selenium (Python)

This repository showcases my automation testing skills using Selenium WebDriver with Python, structured using the Page Object Model (POM) design pattern.

Tech Stack:
- Python
- Selenium WebDriver
- Page Object Model (POM)
- Explicit Wait (WebDriverWait)
- Git & GitHub

Project Overview:
This project automates a complete end-to-end user flow:
- Login to website
- Select product
- Add to cart
- Proceed to checkout
- Select country from dynamic dropdown
- Complete purchase
-Validate success message

The framework is designed with clear separation of responsibilities to ensure maintainability and reusability.

Project Structure:
- loginpage.py – Handles login actions
- product_page.py – Handles product selection and checkout
- main.py – Controls the full E2E automation flow
- README.md – Project documentation
